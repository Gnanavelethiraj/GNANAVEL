The BMW R 1250 GS is a versatile adventure motorcycle manufactured
by BMW Motorrad in Berlin, Germany. Here are some key details about it:
© Engine: The R 1250 GS features a 1,254 cc (76.5 cu in) two-cylinder boxer engine
with four valves per cylinder and variable valve timing (VVT).
This innovative ShiftCam engine design provides smooth power delivery and
excellent performance.
© Features: LED flashing turn indicators come standard.
Optional features include Cruising Light, handlebar risers (30 mm),
and an enduro package for off-road adventures.
Comfortable heated seat.
Adaptive headlight with LED technology.
Ride Modes Pro and ECO Modes.
Dynamic Traction Control (DTC) and ABS Pro.
Connectivity with a TFT display and USB remote positive terminal.
